软技能:代码之外的生存之道-理财篇

Posté le 2018-11-18

概览

由于软件开发人员的职业特性，很难有人说能够干一辈子的程序员，这个现象在国内尤为严重（毕竟人多，竞争压力大）。所以，提早开始理财，早作打算，以免到时没有抵御任何风险的能力。

1. 薪水的支配

对大多数程序员而言，都是通过工作来换取固定的报酬的，尤其是初入职场的新人。每月基本固定的工资，但是不同的人有不同的消费习惯，最后会导致两个工资差不多的人，在工作一年之后会有完全不一样的结果。

1.1 每月硬存

这是最简单的方式，每个月在发薪水的第一时间，将部分工资直接存起来。

  • 我这边直接是存在了苏宁金融的易付宝-零钱宝这个产品中，类似于支付宝但是利率相对较高。
  • 家里条件还可以的，可以通过父母付个首付，然后买个住房，然后每个月自己还款。这样做既省了租房的费用，也变相的将钱存成了房产。毕竟钱随着时间的延长一般是会慢慢的贬值的。
  • 还有一种方式就是每个月投入固定的钱到风险较高的基金或者股票中去。这个方式风险较大，适用于了解过一定金融知识的人。

    1.2 每月记账

    每个月的每一笔账都需要去记，不管是收入还是支出，养成记账的习惯。
  • 知道自己的资金动向。
  • 看看哪些是必要的支出，哪些是进阶支出。
  • 根据支出情况，合理规划自己每个月的薪水。
  • 如果可能的话，尽量不要做月光族。这样你在人生online这个游戏里面的可操作性就太低了。

    2. 薪资谈判

    当我看这本书时，我的薪资已经被安排了。。。目测是比原定的的薪水低了许多。（年轻，无知的我，每每念及此事，心口总是隐隐作痛）

    2.1 熟人介绍

    如果有看过工作篇中的职位推荐，应该还记得，有熟人推荐（管事的最好）的话，更容易获得较高的薪资。

    2.2 提高知名度

    如果你在业内很有名气，你可以开出的价格自然会水涨船高。（原理的话，请参考流量明星与老戏骨，同一集电视剧的报价）。提高知名度的方式在工作篇中也有讲过，这边再过一遍：写博客，写书，写教程，在大会上演讲，制作教学视频等，说到底，就是自我营销。

    2.3 其他技巧

  • 不要先出价，先出的价格给对面透露了更多的信息，让自己处于相对被动的信息（就好像武侠小说里面，先出手的一般都凉了；就像皇室战争里面，高手对决，一般都是等对方先出手）
  • 拿到offer的时候，再提一波加薪（这个方法我没用过，不作评价）
  • 知己知彼，方能百战不殆。最简单的也是最容易被忽视的，多了解这个职位需要什么，看自己有什么，然后好好准备（做的同样的事情，有些人就是能说的很高大上。请参考各种手机的新品发布会）。很简单的道理，讲师我就没用过，皮。。。

    3. 房产投资

    房产投资这块我也是云玩家，学过，见过，听说过，就是没玩过。

    3.1 住房投资

    该书作者力荐！！！他可能眼光比较好，然后运气确实不错，买的房子都顺利的租出去了，然后以租养贷，白嫖住宅楼一栋。（可能跟美国的贷款政策也有关系）

    3.2 店铺投资

    这个可能在国内更适合一点吧？毕竟店铺不像现在的住房，越多套房交税越多（在此前十年房产貌似一直都是稳赚不赔的投资）。

    3.3 公寓投资

    云玩家听说过的又一种投资方式，在一线城市的黄金地段屯公寓，然后租给白领住，以租养贷，白嫖公寓一套。有一次去青岛玩，整个一栋楼的公寓基本都被买下来，然后租给来旅游的游客，平常200~300，旺季600-1000都有（每天）。
